-----------------------------------
Cartoon Comic Skyboxes Pack 2
-----------------------------------

This package contains 6 high-resolution skyboxes for Cartoon/Comic games. There are 3 types of sky (day, night and sunrise/sunset) in this package. Each skybox has two styles for you to choose to match the style of your game.

Texture size:
6 x 2048 x 2048 for each skybox.

Package comes with sample script and demo scenes that help you quickly preview the skyboxes. The demo scenes are located in the folder "\Assets\Cartoon Comic Skyboxes Pack 2\Demo Scenes\".

This project has been tested on unity 2017.4.1/ 2018.4.3/ 2019.2.12.
